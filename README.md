# Todo Apps
aplikasi ini dapat digunakan untuk membuat rencana untuk kegiatan yang akan dilakukan, anda bisa melakukan ceklis pada kegiatan atau rencana yang telah selesai dilakukan. aplikasi ini dibuat menggunakan react dan di styling menggunakan tailwindcss.

![references](./src/assets/img/references.jpg)

## cara menggunakan
1. download semua file di repo ini 
2. buka terminal
3. ketikan npm run dev
